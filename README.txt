# Brick & Beyond LLC Website

## Files included
- index.html
- styles.css
- script.js

## How to run
Open `index.html` in your browser.

## How to put online
Upload all 3 files to your hosting provider, Netlify, GitHub Pages, or Hostinger public_html folder.
